Автоматическая кормушка (Arduino)

Проект автоматизированного устройства для выдачи корма по расписанию. Устройство использует модуль реального времени для отслеживания времени и сервопривод для механической выдачи порции.
Основные возможности

    Установка расписания: Пользователь может задать время кормления (часы и минуты) с помощью внешней клавиатуры (4x4).  

    Автономная работа: Время кормления сохраняется в энергонезависимую память (EEPROM), поэтому настройки не сбрасываются после выключения питания.  

    Индикация: Текущее время и дата отображаются на LCD-дисплее (16x2).  

    Механизм выдачи: Использование сервопривода для точного дозирования корма.  

Аппаратная часть

    Контроллер: Arduino (Mega/Uno).

    Часы реального времени (RTC): Модуль DS1302.  

    Ввод данных: Матричная клавиатура 4x4.  

    Вывод информации: LCD-дисплей 16x2 с I2C-адаптером.  

    Исполнительный механизм: Сервопривод.  

Управление

    Просмотр: В обычном режиме на экране отображается текущее время и дата.  

    Настройка: Нажатие кнопки (пин A3) переводит устройство в режим установки времени.  

    Ввод: Введите время в формате ЧЧ:ММ с клавиатуры и нажмите клавишу D для сохранения настроек.
